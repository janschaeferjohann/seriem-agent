This folder contains the XSD schema definition for XML content contributed via the DataContent property of a model,
as described in M/TEXT TONIC Text Administration manual chapter "Passing formatted texts".

Root elements of the schema are defined in RootSIMPLEDOMElements.xsd - it is either a ContainerPart or a ParPart.

In addition, the subfolder Charts contains a XSD schema for TONIC Chart images. Charts have their own 
namespace and they are not directly referenced from the main schema - see the description of the InlineImageData element
in AbstractImage.xsd.

-----------------------------------------------------------------------------------------------
Dieser Ordner enthält die XSD-Schemadefinition für XML-Inhalte, die über die DataContent-Eigenschaft eines Modells eingebracht werden,
wie im M/TEXT TONIC Text Administration Handbuch Kapitel „Übergabe formatierter Texte“ beschrieben.

Root-Elemente des Schemas sind in RootSIMPLEDOMElements.xsd definiert - es ist entweder ein ContainerPart oder ein ParPart.

Darüber hinaus enthält der Unterordner Charts ein XSD-Schema für die TONIC Diagramme. Diagramme haben ihren eigenen 
Namespace und werden nicht direkt vom Hauptschema referenziert - siehe die Beschreibung des InlineImageData-Elements
in AbstractImage.xsd.
